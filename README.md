## Digital Twin RAG Interview Agent

AI-powered interview agent using Retrieval-Augmented Generation (RAG) to 
conduct grounded, factual interview simulations. Built during a 10-week 
AI Data Analyst industry internship (Employment Advantage x AusBiz 
Consulting).

**Role:** Scrum Leader, 6-person team. Coordinated Git workflows, resolved 
merge conflicts, managed pull requests across 183+ commits, and 
maintained technical documentation.

| Component | Technology |
|---|---|
| Vector Database | Upstash Vector |
| LLM Inference | Groq (LLaMA 3.1) |
| Agent Mode | Claude Sonnet |
| Tool Calling | MCP (Model Context Protocol) |
| Frontend | Next.js, TypeScript |
| Deployment | Vercel |

[Repository](https://github.com/ranneapic2/digital-twin-Team_2) · [Live Demo](https://digital-twin-team-2.vercel.app)
